## 🚀 SOBRE O PROJETO

Este projeto foi desenvolvido com a proposta de mostrar, de forma **bonita, moderna e inspiradora**, como é possível criar um site impactante usando apenas **HTML com CSS embutido no próprio arquivo**.

A página foi pensada para funcionar como uma **landing page informativa** sobre desenvolvimento de sites em HTML, destacando benefícios, etapas do processo, exemplos de elementos visuais e boas práticas de composição.

Tudo foi concentrado em **um único `index.html`**, sem dependências externas, sem frameworks e sem JavaScript.

## 🌍 PUBLICANDO NO GITHUB PAGES

Se quiser deixar seu site online pelo GitHub Pages:

1. Faça upload do `index.html` e deste `README.md` no repositório.
2. Vá em **Settings**.
3. Acesse **Pages**.
4. Em **Source**, escolha a branch principal.
5. Salve as alterações.
6. O GitHub irá gerar um link público para o seu site.
